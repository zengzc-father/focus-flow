# Focus Flow UI/UX 设计规范

## 设计哲学

> **简洁、温暖、专业**

参考 DeepSeek、Claude App 等优秀 AI 产品，打造清爽无干扰的用户体验。

### Chat界面设计要点

**1. 消息气泡**
```
Agent气泡:
- 背景: #F0FDF9 (极浅青)
- 圆角: 16px (左上圆角小8px，区分用户)
- 内边距: 16px
- 最大宽度: 85%
- 左侧头像: 🤖

用户气泡:
- 背景: #00BFA5 (主色)
- 文字: 白色
- 圆角: 16px (右上圆角小8px)
- 内边距: 16px
- 最大宽度: 85%
- 右对齐
```

**2. 快捷指令栏**
```
位置: 输入框上方
样式: 横向滚动，胶囊按钮
按钮: 圆角20px，浅灰背景，深灰文字
选中: 主色背景，白色文字
```

**3. 输入框**
```
高度: 48-56px (根据内容自适应)
圆角: 24px (胶囊形)
背景: #F3F4F6
边框: 无 (聚焦时显示主色边框)
内边距: 16px水平
语音按钮: 右侧圆形，24x24
```

**4. 功能卡片**
```
规则卡片:
- 背景: 白色
- 边框: 1px #E5E7EB
- 圆角: 12px
- 图标: 左侧
- 操作按钮: 右侧
```

## 交互细节

### 1. 消息动画
```
发送消息:
- 用户消息: 从右侧滑入，100ms
- Agent思考: 显示"..."打字指示器
- Agent回复: 从左侧淡入，200ms

新规则卡片:
- 从底部滑入 + 淡入
- 轻微弹性效果
```

### 2. 触摸反馈
```
按钮点击:
- 缩放: 0.97
- 背景变暗 5%
- 持续时间: 100ms

卡片点击:
- 轻微下沉 (translateY: 2px)
- 阴影加深
```

### 3. 加载状态
```
Agent思考中:
- 三个点依次跳动
- 颜色: 主色
- 下方文字: "Focus 正在思考..."

数据加载:
- 骨架屏 shimmer 效果
- 或圆形进度指示器
```

### 4. 空状态设计
```
无规则:
- 插图: 简单的线条画
- 文字: "还没有规则"
- 按钮: "让 Focus 帮你创建一个"

无数据:
- 文字: "今天还没有使用记录"
- 提示: "放下手机，享受生活~"
```

## 动效规范

### 页面切换
```
类型: 淡入 + 轻微滑动
持续时间: 200-300ms
缓动: ease-out
```

### 列表加载
```
类型: 依次淡入
延迟: 每个item 50ms
持续时间: 200ms
```

### 数字变化
```
类型: 数字滚动
持续时间: 600ms
缓动: ease-out
```

## 响应式设计

### 小屏手机 (<375px)
```
字体缩小 10%
间距缩小 20%
卡片圆角: 12px
```

### 大屏手机 (>414px)
```
内容最大宽度: 400px
居中显示
```

---

## 无障碍设计

```
颜色对比度: 至少 4.5:1
触摸目标: 至少 44x44px
支持屏幕阅读器
支持字体缩放
支持减少动画
```

---

## 设计原则总结

1. **少即是多**: 每页只展示核心信息
2. **温暖亲切**: 像朋友对话，不机械
3. **即时反馈**: 操作有响应，状态可见
4. **一致性**: 设计语言统一
5. **可预期**: 用户知道会发生什么
