
- 预计时间：5-15 分钟

### 步骤 2: 配置环境变量 (自动)
- 添加 `C:\src\flutter\bin` 到 PATH

### 步骤 3: 配置国内镜像 (自动)
```bash
flutter config --set PUB_HOSTED_URL=https://pub.flutter-io.cn
flutter config --set FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
```

### 步骤 4: 验证安装
```bash
flutter --version
flutter doctor
```

### 步骤 5: 接受 Android 许可证
```bash

```

### 步骤 6: 构建 APK
```bash
cd C:\Users\21498\focus-flow-app
flutter pub get

```
